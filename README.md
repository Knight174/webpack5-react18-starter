# webpack5-react18-starter

A development template (react18 + ts) by webpack5.

## Installation

```pnpm i```

## Usage
### Development server
```pnpm dev```

### Production build
```pnpm build```

### Features

- [Babel](https://babeljs.io/) Babel is a JavaScript compiler.
- [Sass](https://sass-lang.com/) CSS preprocessor
- [PostCSS](https://postcss.org/) A tool for transforming CSS with JavaScript
- [React18](https://react.dev/) The library for web and native user interfaces
- [TypeScript](https://www.typescriptlang.org/) TypeScript is JavaScript with syntax for types.
- [EditorConfig](https://editorconfig.org/) EditorConfig helps maintain consistent coding styles for multiple developers working on the same project across various editors and IDEs.
